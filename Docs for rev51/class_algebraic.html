Detailed Description

A class representing an algebraic number.

Internally, the number is represented by a polynomial and a closed interval on which the polynomial has one and only one root, which is the number. Note that the interval can be be a single point (i.e. [a, a]). Internally, the polynomial is represented by a GiNaC::ex object which has copy-on-write semantics.
