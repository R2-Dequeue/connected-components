Detailed Description

The cyclical algebraic decomposition of a set of polynomials.

This object presents an interface for determining the partitions of the plane made by two polynomials. For example [x+y, x-y] creates four partitions, and [x^2+y^2-1, x^2+y^2-4] creates three partitions.
